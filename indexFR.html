HandyMiner King Cobra - Guide de démarrage rapide
  • Telecharger HandyMiner Version ZIP ou Telecharger HandyMiner Version Tarball
  • Prérequis:
    • Installer nodeJS
    • (windows uniquement) Installer git bash dans le répertoire Programmes/Git
  • Exécution:
    • Windows: double cliquer 'dashboard.windows.bat'
    • Linux: double cliquer 'dashboard.sh'
    • Mac: double cliquer 'dashboard.mac.command'
    • CLI: ou
  • Configuration: La première exécution nécessite la configuration matérielle
    • Selectionner les GPUs
    • Choisir un mode ( solo | pool )
    • Intensité: 11 pour un mode optimale
    • Difficulté de la pool: -1 pour une difficulté variable
    • Hébergeurs de Pool:
      • f2pool port:
        identifiant: adresse.NomduMineur
        mot de passe: au choix
      • 6block port:
        identifiant: NomduMineurInscrit
        mot de passe: au choix
      • PoolFlare port:
        identifiant: adresse.NomduMineur
        mot de passe: au choix
      • hnspool port:
        identifiant: NomUtilisateurInscrit
        mot de passe: MotdePasseInscrit
  • Porte-monnaies:
    • Bob Wallet (Interface Graphique, contrôlé par l’utilisateur)
